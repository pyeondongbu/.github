# 🎬 편동부 (PyeongDongBu)

편동부는 영상 편집자들을 위한 구인구직 정보를 한눈에 모아보고 공유하는 플랫폼입니다.

## 주요 기능
- 편집자 구인 정보 수집 및 공유
- 맞춤형 구인 정보 검색
- 구인/구직 게시글 작성 기능
